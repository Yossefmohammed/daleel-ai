Wasla Solutions Support FAQ

1. What can Wasla AI support?
Wasla AI can answer questions about Wasla Solutions' services, digital capabilities, and document-based knowledge. It is not a live support agent, but it can guide you using the information provided in the knowledge base.

2. Which document types are supported?
We support PDF, TXT, and MD files in the docs folder. Add your support articles, FAQs, and product details there.

3. How do I ask a good support question?
Ask clearly and include the topic you are interested in. For example:
- "What does Wasla offer for website projects?"
- "How do I create a knowledge base document?"
- "What should I do if I need support?"

4. What happens if the information is missing?
If the assistant does not find the answer in the knowledge base, it will say so and ask if you want more help or if you want to contact support.

5. Who should use this assistant?
This assistant is designed for customers and team members who need quick answers from Wasla's internal knowledge documents.

6. How to improve answers?
Add clear, structured support content to the docs folder. Use headings, short paragraphs, and specific bullet lists.
